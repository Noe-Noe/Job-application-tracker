## Responsive Design

### Desktop (> 768px)
- 4-column stats grid
- Full filter bar with all options visible
- 2-column event details in cards

### Mobile (< 768px)
- 1-column stats grid (stacked)
- Simplified filter bar (stacked)
- 1-column event details
- Touch-friendly buttons
