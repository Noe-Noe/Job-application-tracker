# Browser Extension Quick Start

## Installation

1. Load the extension in your browser (Chrome/Edge/Firefox)
2. Click the extension icon
3. Go to your Job Tracker Settings page
4. Copy the API Endpoint and API Token
5. Paste them into the extension popup
6. Click "Save Configuration"
7. Click "Test Connection"

## Usage

Visit any job posting page → Click "Save to Job Tracker" button → Done!

See [EXTENSION_README.md](../EXTENSION_README.md) for full documentation.
